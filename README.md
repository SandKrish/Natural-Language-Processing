# Natural Language Processing – Disaster Tweet Classification

This repository contains a notebook that analyzes **disaster and non-disaster tweets using Natural Language Processing (NLP)** and a **Long Short-Term Memory (LSTM) model**.

## Notebook

**nlp-with-disaster-tweets-using-lstm.ipynb**

The notebook includes:

* Introduction to Natural Language Processing (NLP)
* Text preprocessing and Bag of Words representation
* Explanation of Long Short-Term Memory (LSTM) networks
* Classification of tweets as disaster or non-disaster

## Technologies Used

* Python
* Pandas
* NumPy
* NLP techniques
* LSTM (Deep Learning)
* Jupyter Notebook
